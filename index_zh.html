让 AI 通过真实 Unreal Editor 命令编辑 UMG。

UMG MCP 把 Unreal UMG 制作暴露为结构化 MCP 工具。Agent 可以读取 Widget Tree、编辑布局、连接 Blueprint、创建 UI 材质并添加动画，而不是靠截图猜测。

  2. 公开文档站归属于开源仓库。
  3. Fab 包只是可选打包分发渠道。

Agent 可以编辑什么

结构化访问编辑器，而不是盲目自动点击。

WidgetUMG 布局和属性

创建控件、读取树结构、设置 Slot、重排子节点、替换子树，并用 JSON 做导出和回写。

Blueprint图表操作

创建节点、连接 Pin、绑定 Widget 引用，并用 BlueCode 做紧凑图表编辑。

MaterialUI 材质和 HLSL

创建材质资产、设置参数，并执行支持的 HLSL 相关材质操作。

AnimationUMG 动画轨道

通过 Sequencer 相关工具添加 Render Transform、Opacity、Color 和 Timing 关键帧。

Editor本机桥接

外部 MCP 客户端通过 127.0.0.1:55557 的本机桥接与 Unreal Editor 通信。

Protocol可测试的命令表面

协议文档和 API 测试与源码一起维护，方便公开复现和修复。

快速开始

先安装开源插件；只有需要市场安装包时再选择 Fab 分发。

1. 克隆

把仓库克隆到 YourProject/Plugins/UmgMcp。

2. 编译

打开 Unreal Engine 5.8，让 Unreal Build Tool 编译插件。

3. 连接

配置 MCP 客户端运行 Resources/Python/UmgMcpServer.py。

4. 验证

运行 APITest 检查，再做编辑器冒烟测试。
