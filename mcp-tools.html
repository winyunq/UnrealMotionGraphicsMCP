Fab 内置工具面说明

Fab 内置 Agent 使用“筛选工具面”，不是另一个 89 项清单

这是旧文档最容易混淆的地方。

Tool Mode

UMG、Blueprint、Animation、Material 模式会按任务选择允许暴露给内置 Agent 的工具，降低误调用和上下文噪声。

task_begin / task_end

它们用于 Fab 的 Agent/Task 交接和 UI 展示，是编排伪工具，不属于外部 MCP Server 的 89 工具。

原始 C++ CommandType

底层派发器可能支持额外帮助命令；只有被 UmgMcpServer.py 注册后，才计入公开 MCP 工具面。
