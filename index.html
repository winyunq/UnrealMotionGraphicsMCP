AI-assisted UMG editing through real Unreal Editor commands.

UMG MCP exposes Unreal UMG authoring as structured MCP tools. Agents can read widget trees, edit layouts, wire Blueprint logic, create UI materials, and add animation without guessing from screenshots.

What agents can edit

Structured access to the editor, not blind UI automation.

WidgetUMG layout and properties

Create widgets, inspect trees, set slots, reorder children, replace subtrees, and round-trip UI as JSON.

BlueprintGraph operations

Create nodes, connect pins, bind widget references, and use BlueCode for compact graph edits.

MaterialUI materials and HLSL

Create material assets, set parameters, and work with supported HLSL-oriented material operations.

AnimationUMG animation tracks

Add keyframes for render transform, opacity, color, and timing through Sequencer-oriented tools.

EditorLocal bridge

External MCP clients communicate with Unreal Editor through the local bridge on 127.0.0.1:55557.

ProtocolTestable command surface

Protocol markdown and API tests live with the source so issues can be reproduced and fixed openly.

Quick start

Install the open-source plugin first; add Fab packaging only if you need that distribution path.

1. Clone

Clone this repository into YourProject/Plugins/UmgMcp.

2. Build

Open Unreal Engine 5.8 and let Unreal Build Tool compile the plugin.

3. Connect

Configure your MCP client to run Resources/Python/UmgMcpServer.py.

4. Verify

Run the APITest checks, then test an editor smoke workflow.

Open-source first
